# Настройка аналитики для сайта Два-Кадра

## Google Analytics (GA4)

### Шаг 1: Создание аккаунта Google Analytics
1. Перейдите на https://analytics.google.com/
2. Нажмите "Начать измерение"
3. Создайте новый аккаунт со следующими параметрами:
   - **Название аккаунта**: Два-Кадра
   - **Название свойства**: Два-Кадра Studio
   - **Часовой пояс**: Europe/Moscow (UTC+3)
   - **Валюта**: RUB (Российский рубль)

### Шаг 2: Получение ID отслеживания
1. После создания свойства вы получите **ID измерения** (формат: G-XXXXXXXXXX)
2. Скопируйте этот ID

### Шаг 3: Интеграция в сайт
1. Откройте файл `client/index.html`
2. Найдите строку `<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>`
3. Замените `G-XXXXXXXXXX` на ваш ID измерения
4. Найдите строку `gtag('config', 'G-XXXXXXXXXX', {`
5. Замените `G-XXXXXXXXXX` на ваш ID измерения

### Что отслеживается:
- Просмотры страниц
- События (клики по кнопкам, заполнение форм)
- Источники трафика
- Поведение пользователей
- Конверсии

## Яндекс.Метрика

### Шаг 1: Создание счетчика
1. Перейдите на https://metrica.yandex.ru/
2. Нажмите "Добавить счетчик"
3. Введите параметры:
   - **Название счетчика**: Два-Кадра
   - **Сайт**: dva-kadra-studio.manus.space
   - **Часовой пояс**: Europe/Moscow (UTC+3)
   - **Тип счетчика**: Стандартный

### Шаг 2: Получение номера счетчика
1. После создания вы получите **номер счетчика** (формат: 1234567890)
2. Скопируйте этот номер

### Шаг 3: Интеграция в сайт
1. Откройте файл `client/index.html`
2. Найдите строку `ym(XXXXXXXXXX, "init", {`
3. Замените `XXXXXXXXXX` на номер вашего счетчика
4. Найдите строку `<img src="https://mc.yandex.ru/watch/XXXXXXXXXX"`
5. Замените `XXXXXXXXXX` на номер вашего счетчика

### Что отслеживается:
- Посещения и просмотры
- Источники трафика (поисковые системы, соцсети, прямые переходы)
- Поведение пользователей (клики, скролл, время на странице)
- Воронки конверсии
- Карта кликов (Heatmap)
- Сегментация аудитории
- UTM-параметры

## Дополнительные рекомендации

### UTM-параметры
Используйте UTM-параметры для отслеживания источников трафика:
- `utm_source` - источник (vk, telegram, rutube, tiktok, youtube)
- `utm_medium` - тип трафика (social, video, direct)
- `utm_campaign` - название кампании (например: winter_promo_2025)
- `utm_content` - дополнительная информация (например: banner_1)
- `utm_term` - ключевое слово

**Пример ссылки:**
```
https://dva-kadra-studio.manus.space/?utm_source=vk&utm_medium=social&utm_campaign=winter_promo&utm_content=banner_1
```

### Проверка установки

**Google Analytics:**
1. Откройте сайт в браузере
2. Откройте DevTools (F12)
3. Перейдите на вкладку "Network"
4. Поищите запросы к `google-analytics.com` или `googletagmanager.com`
5. Если запросы есть - GA4 установлена корректно

**Яндекс.Метрика:**
1. Откройте сайт в браузере
2. Откройте DevTools (F12)
3. Перейдите на вкладку "Console"
4. Введите `ym` и нажмите Enter
5. Если функция определена - Метрика установлена корректно

### Рекомендуемые события для отслеживания

1. **Клик по кнопке "Записаться на занятие"**
   - Событие: `sign_up`
   - Параметры: программа, источник

2. **Просмотр видео "Ежовости"**
   - Событие: `video_view`
   - Параметры: номер видео, длительность

3. **Клик по ссылке социальной сети**
   - Событие: `social_link_click`
   - Параметры: платформа (vk, telegram, youtube и т.д.)

4. **Просмотр галереи**
   - Событие: `gallery_view`
   - Параметры: категория (filming, studio, editing и т.д.)
